# Telegram Planner Bot

Бот для планирования задач и саморазвития в Telegram.

## Функции

- 📅 **Планирование** - создание задач с напоминаниями по времени
- 👤 **Профиль** - статистика выполнения задач и оценка прогресса
- 🧠 **Тренинг** - ежедневные упражнения для развития мышления
- ℹ️ **Инфо** - информация о боте

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

5. Откройте `.env` и вставьте токен:
```
BOT_TOKEN=ваш_токен_здесь
ADMIN_ID=ваш_telegram_id
```

## Запуск

```bash
python bot.py
```

## Добавление картинок

Для добавления картинок в сообщения:

1. Создайте папку `images/` в корне проекта
2. Добавьте туда свои картинки (например: `main_menu.jpg`, `profile.jpg` и т.д.)
3. Раскомментируйте строки с `FSInputFile` в `bot.py` и укажите пути к картинкам

Пример:
```python
photo = FSInputFile("images/main_menu.jpg")
await message.answer_photo(photo, caption="Текст", reply_markup=keyboard)
```

## Использование

1. Найдите бота в Telegram по username (который вы указали при создании)
2. Отправьте команду `/start`
3. Используйте кнопки для навигации по разделам

### Добавление задачи в планирование

1. Нажмите "📅 Планирование"
2. Нажмите "➕ Добавить задачу"
3. Отправьте сообщение в формате:
```
Текст задачи
14:30
```

Бот отправит напоминание в указанное время и будет повторять его каждые 5 минут (до 5 раз), пока вы не нажмете "Начинаю!"

## База данных

Бот использует SQLite. База данных создается автоматически при первом запуске в файле `bot_database.db`.

## Примечания

- Напоминания работают по серверному времени. Для корректной работы с часовыми поясами можно доработать функцию `check_schedule()`
- Максимум 10 задач в планировании на пользователя
- Упражнения для тренинга обновляются каждый день
- Статистика профиля обновляется в реальном времени
