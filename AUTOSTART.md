# Автозапуск бота

## Способ 1: Добавить в автозагрузку Windows

1. Нажми `Win + R`
2. Введи: `shell:startup`
3. Скопируй файл `start_bot_hidden.vbs` в открывшуюся папку
4. Готово! Бот будет запускаться при старте Windows в фоновом режиме

## Способ 2: Через Планировщик заданий Windows (более надежный)

1. Нажми `Win + R`, введи `taskschd.msc`
2. Справа нажми "Создать простую задачу"
3. Имя: "Telegram Bot"
4. Триггер: "При входе в Windows"
5. Действие: "Запустить программу"
6. Программа: `C:\scripts\telegram_planner_bot\start_bot_hidden.vbs`
7. Готово!

## Способ 3: Запуск вручную

Просто запусти файл `start_bot_hidden.vbs` двойным кликом - бот запустится в фоне.

## Как остановить бота?

1. Открой Диспетчер задач (Ctrl + Shift + Esc)
2. Вкладка "Подробности"
3. Найди процесс `python.exe` с командной строкой `bot.py`
4. Завершить процесс

## Проверка работы

Бот работает, если:
- В Telegram бот отвечает на команды
- В диспетчере задач есть процесс python.exe

## Логи

Для просмотра логов запусти `start_bot.bat` - откроется окно консоли с выводом.
